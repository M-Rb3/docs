## Ecosystem Flywheel

<Steps>
  <Step title="Users engage → fees generated">
    Active users drive ecosystem activity and revenue generation
  </Step>
  
  <Step title="Revenue → buybacks + staking pools">
    Platform revenue funds token buybacks and ecosystem development
  </Step>
  
  <Step title="Utility → demand ↑ token value ↑">
    $CENT token provides value through discounts, rewards, and governance
  </Step>
  
  <Step title="Rewards → user retention">
    Better rewards drive user retention and long-term commitment
  </Step>
  
  <Step title="DAO allocates treasury → devs + merchants">
    Community-driven growth expands ecosystem reach and capabilities
  </Step>
</Steps>
