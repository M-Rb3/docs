## Flywheel Overview

<Card title="Self-Reinforcing Growth" icon="sync" horizontal>
  The Cent ecosystem flywheel creates a positive feedback loop where user
  engagement, platform revenue, token value, and ecosystem growth reinforce each
  other to drive sustainable long-term growth.
</Card>

## The Flywheel Mechanism

<Steps>
  <Step title="User Engagement">
    Users engage with Cent services (FX, Pay, Exchange) generating transaction
    volume and platform usage.
    <Check>Active user base drives ecosystem activity and revenue.</Check>
  </Step>

  <Step title="Revenue Generation">
    Increased usage generates fees and revenue for the platform across all
    services.
    <Check>
      Higher revenue enables more token buybacks and ecosystem development.
    </Check>
  </Step>

  <Step title="Token Value Appreciation">
    Revenue-driven buybacks and utility demand increase $CENT token value.
    <Check>
      Higher token value attracts more users and increases engagement.
    </Check>
  </Step>

  <Step title="Enhanced Rewards">
    Higher token value and increased staking rewards attract more participants.
    <Check>Better rewards drive user retention and long-term commitment.</Check>
  </Step>

  <Step title="Ecosystem Growth">
    DAO governance allocates treasury resources for development and
    partnerships.
    <Check>
      Community-driven growth expands ecosystem reach and capabilities.
    </Check>
  </Step>
</Steps>

<Steps>

    <Check>Real utility drives organic demand for tokens.</Check>
  </Step>
