## Product Roadmap

<Steps>
  <Step title="Q4 2025 (Phase 1)">
    - Cent Token Platform website
    - Retail dashboard
    - Corporate dashboard (basic)
    - Admin panel v1
  </Step>

  <Step title="Q1 2026 (Phase 2)">
    - Deposits, withdrawals, swaps
    - Corporate reporting
    - Expanded admin tools
  </Step>

  <Step title="Q2 2026">
    - Multi-token support
    - MPC wallet integration
    - Loyalty + staking dashboard
  </Step>

  <Step title="Q3 2026">
    - Mobile apps (iOS + Android)
    - Merchant POS + NFC hardware
  </Step>

  <Step title="2027+">
    - DAO governance portal
    - NFT loyalty tiers
    - CentChain (modular L2)
    - Regional expansion: LatAm, Africa, SEA
  </Step>
</Steps>
